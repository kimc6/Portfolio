My work with Robotics!

We programmed Arduino robots to move around obstacles by using whisker wires! If the left whisker wire was touched, the robot would back up and turn clockwise by 60 degrees. If the right wire was touched, the robot would back up and turn counter clockwise by 60 degrees. If both wires were touched, then the robot would back up and turn clockwise by 120 degrees. It was SO COOL using whisker wires because the values read by the whisker wires were in binary code. I hope to do more!
